
# Hangz 🗺️

Hangz - find the spot, skip the stress 

Hangz is een interactieve webapplicatie waarmee je snel en eenvoudig leuke hangspots, eetplekken en activiteiten kunt ontdekken in de regio die jij wilt. Klik op de kaart, voeg je favoriete plekken toe en beoordeel ze met sterren. Alles in een stijlvol donker design met oranje accenten.

![Hangz screenshot](Screenshot%202026-03-01%20204208.png) <!-- Voeg later een screenshot toe -->

 ✨ Features

- Interactieve kaart – Gebaseerd op Leaflet met een donkere stijl.
- Markers voor categorieën – Park, gym, restaurant, activiteit, muziekschool, pedicure.
- Filteren – Bekijk alleen de categorieën die jij interessant vindt.
- Zoeken – Vind spots op naam via de zoekbalk.
- Spots toevoegen – Klik op de kaart, vul een formulier in en voeg je eigen plek toe.
- Beoordelen – Geef 1 tot 5 sterren per spot; het gemiddelde wordt automatisch berekend.
- Gebruikersprofiel – Stel een gebruikersnaam in en bekijk je eigen toegevoegde spots.
- Lokale opslag – Spots en gebruikersnaam worden opgeslagen in `localStorage` (later vervangen door Supabase).
- Responsive design – Werkt op desktop, tablet en mobiel.

 🛠️ Gebruikte technologieën

- HTML5 – Semantische structuur
- CSS – Moderne styling met variabelen, flexbox, grid en animaties
- JavaScript – Volledige functionaliteit, modulaire opzet
- Leaflet – Interactieve kaart met OpenStreetMap
- Font Awesome – Iconen
- Google Fonts – Inter lettertype
- Supabase (in de toekomst) – Backend en database
